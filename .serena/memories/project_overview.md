# Zona English Landing Page - Project Overview

## Purpose
This project is a modern, responsive landing page for **Zona English**, an English learning institution in Indonesia. The page serves as the primary marketing tool to attract students aged 3-25+ years for English language courses.

## Target Audience
- Primary: Students and parents looking for English language education
- Geographic focus: Kolaka, Makassar, Kendari (Indonesia) + online learners
- Age groups: 3-6 (Kid Class), 7-11 (Junior Class), 12-17 (Teen Class), 18-25+ (Adult Class)

## Key Features Highlighted
- NBSN (Neuron, Biomolecular, Sensory, Nature) teaching method
- AI Coach integration
- Small class sizes
- Project-based learning
- Reward system
- Multi-location availability (offline + online)

## Business Context
Part of a larger educational ecosystem including:
- Hira Space, Hira Academy
- Docterbee
- Freshskill.my.id
- ZonaEnglish App
- ZE Camp

## Design Goals
- Clean, modern, and 100% responsive design
- Single-page application approach
- Fast loading times
- Conversion-optimized with multiple CTAs
- Indonesian language content with English learning focus
